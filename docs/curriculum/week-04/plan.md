A: A common mistake is to use confidence intervals mechanically while ignoring assumptions, data quality, or market frictions. In finance, even a mathematically correct method can fail if the data are noisy, the sample is unstable, or the implementation is unrealistic.

If the topic involves prediction or backtesting, I would also guard against leakage and ensure evaluation is aligned with how the model would be used live.

### Q: Why does linear regression matter for a quant trader or quant researcher?
A: Linear regression matters because it changes either signal quality, risk control, implementation quality, or communication with a desk or portfolio manager. A useful quant answer should always connect the concept to a real decision rather than leaving it as theory only.


## Capstone
- Problem statement: Compare a small set of assets, estimate returns and volatility, and design a basic diversified portfolio with a clear risk narrative.
- Data source suggestions: Use Yahoo Finance via yfinance or a cleaned CSV with daily prices for ETFs such as SPY, TLT, GLD, and EEM.
- Implementation steps: Download data, clean missing values, compute returns, summarize descriptive statistics, compare correlations, test a few weight vectors, and write a one-page conclusion.
- Evaluation metrics: Annualized return, annualized volatility, Sharpe ratio, max drawdown, and clarity of written interpretation.
- Common mistakes: Using price levels instead of returns, annualizing incorrectly, ignoring missing data, or treating one short sample as universal truth.
