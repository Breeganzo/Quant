## Global Day-Level Remediation Rule

For every week and every day (1-7):

1. Formula Sheet: select 3 formulas/workflows tied to the day topic.
2. Quiz Q2: ask for Formula 1 from that day topic, not a shared week template.
3. Python Drill: use day-topic code (not shared boilerplate) and require interpretation.
4. Real-World Lab: include one concrete output metric and one failure-mode check.
5. Notebook Labs 1-4: align with day topic and include a robustness note.
