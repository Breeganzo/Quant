## Learning Overview
Portfolio theory is one of the core languages of quant finance. Even if later practice becomes more complex, the risk-return framework remains central.

You should come away knowing not just what the efficient frontier and Sharpe ratio are, but also why naive use can be misleading.

## Core Concepts
### Mean-variance thinking
- Simple explanation: Portfolio theory weighs expected return against risk.
- Technical depth: Risk is often approximated by variance or standard deviation, and diversification enters through covariance.
- Finance example: Two portfolios with the same expected return can differ sharply in volatility.

### Efficient frontier
- Simple explanation: The efficient frontier is the set of portfolios offering the best return for a given level of risk.
- Technical depth: It is generated by optimization under expected-return and covariance assumptions.
- Finance example: A portfolio below the frontier is inefficient because another available portfolio dominates it.

### Sharpe ratio intuition
- Simple explanation: Sharpe ratio measures return per unit of risk.
- Technical depth: It compares excess return to volatility, but it depends on estimated inputs and does not capture every risk type.
- Finance example: A smooth-looking strategy can have a high Sharpe yet still hide tail or liquidity risk.

## Practice Questions With Answers
### Question: What does a high Sharpe ratio suggest?
Answer: It suggests strong average return relative to volatility, but it does not guarantee robustness or protection from tail risk.

### Question: Why is the efficient frontier fragile in practice?
Answer: Because it depends heavily on estimated returns and covariances, which are noisy and unstable.
