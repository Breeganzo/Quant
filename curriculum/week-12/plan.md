## Capstone
- Problem statement: Build a mini option-risk notebook that explains payoffs, Greeks intuition, and scenario behavior for a small set of option positions.
- Data source suggestions: Use toy option parameters, public option chain snapshots if available, or synthetic examples.
- Implementation steps: Define payoff functions, explore parameter changes, simulate simple scenarios, and write a risk memo explaining exposures.
- Evaluation metrics: Correct payoff logic, coherent scenario analysis, clarity of Greek interpretation, and quality of written communication.
- Common mistakes: Confusing payoff with profit, ignoring premium cost, using formulas without understanding what drives sensitivity, and skipping written interpretation.
